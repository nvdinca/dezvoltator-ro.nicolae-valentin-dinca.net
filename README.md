## RO

### Descriere

Website de prezentare pentru dezvoltatori imobiliari din Romania.
Proiect realizat pentru portofoliul meu, creat pentru a demonstra cum un dezvoltator isi poate promova eficient online proiectele rezidentiale.

Aplicatia include prezentarea:
- ansamblurilor rezidentiale
- apartamentelor individuale
- facilitatilor si informatiilor esentiale pentru clienti

Focusul principal este pe:
- structura moderna si clara
- experienta vizuala premium
- pagini gandite pentru conversii (lead-uri)

### Scopul proiectului

Acest proiect este un demo profesional pentru portofoliu, orientat catre clienti din zona imobiliara (rezidential, premium, luxury), pentru a exemplifica:
- landing page-uri de proiect
- pagini dedicate apartamentelor
- formulare de contact simple si eficiente

### Tehnologii folosite

- Next.js
- TypeScript
- Tailwind CSS
- Vercel (deploy)

### Rulare locala

```bash
npm install
npm run dev
```

Aplicatia ruleaza la `http://localhost:3000`.

### Status

Proiect in dezvoltare activa pentru portofoliu.

## EN

### Description

Presentation website for real estate developers in Romania.
This project is part of my portfolio and showcases how a developer can effectively promote residential projects online.

The application includes:
- residential complex presentation pages
- individual apartment pages
- facilities and key project information for potential buyers

Main focus:
- modern and clear structure
- premium visual experience
- conversion-oriented pages (qualified leads)

### Project purpose

This project is a professional portfolio demo aimed at real estate clients (residential, premium, luxury), designed to showcase:
- dedicated project landing pages
- apartment detail pages
- simple and effective contact forms

### Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel (deployment)

### Local development

```bash
npm install
npm run dev
```

The app runs at `http://localhost:3000`.

Project currently in active development for portfolio purposes.
